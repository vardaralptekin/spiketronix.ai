02 — Colour

Deep teal-black, one electric accent

Grounds

Ink (text)

Accent — use sparingly

Contrast rule: teal is an accent, never body text. Set paragraphs in Ink Soft on the dark ground. Teal is legible only at large sizes (titles, big numbers) or as a thin line — never small teal sentences.

05 — Motion

Use the real animations

You have the site’s animations exported as looping .mp4 (and .gif). Drop one onto a dark slide as the hero visual.

Embedding a clip
  • Insert → Video → This Device → pick the .mp4
  • Playback → Start: Automatically, ✓ Loop until Stopped, ✓ Mute
  • Place on #061018 so the clip’s dark background blends seamlessly
  • One moving element per slide — let it be the focus
Clip library

neuron · feram · imc · hero-chip · deployment-cycle

Each is one seamless loop. Prefer .mp4 (small, sharp); .gif only where video isn’t allowed.

Keep transitions restrained: a single Morph or Fade deckwide. No spins, bounces or checkerboards — the animations already carry the energy.

06 — Rules of thumb

Keep it disciplined

Do
  • One idea per slide; one teal highlight per slide
  • Left-align text; anchor to the same margin every slide
  • Big numbers in teal, tabular, with a mono unit label
  • Solid dark grounds; let negative space breathe
  • Hairline dividers in Teal Dark (#006c5c) at low weight
Don’t
  • Set paragraphs or small text in teal or violet
  • Fill a slide with teal — it stops being an accent
  • Use pure black (#000) or white slide backgrounds
  • Drop below 18pt, or use Thin/Light weights
  • Mix in stock icons or fonts outside the two faces
07 — Quick start

Build the master in 5 steps

Set size & background

Design → Slide Size → Widescreen 16:9. Set slide background fill to solid #061018.

Install & embed the fonts

Install Space Grotesk + Space Mono. Then File → Options → Save → Embed fonts in the file.

Set theme colours

Design → Variants → Colours → Customize. Text/Background = Ink #f4f8f7 / Base #061018; Accent 1 = Teal #00d3b1; Accent 2 = Violet #6954ff.

Lay out the master slides

In Slide Master, build the five layouts above — eyebrow placeholder top-left, title block, footer with logo + page number.

Add motion where it earns its place

Insert the exported .mp4 loops on media slides; set Autoplay + Loop + Mute. One clip per slide.
